
# Arabic SSL OCR Final Project

## Project Title
Self-Supervised OCR for Low-Resource Languages: Reducing Labeled Data Requirements for Arabic Scripts

## Project Goal
This project investigates whether self-supervised learning can reduce the amount of labeled Arabic OCR data needed while maintaining acceptable recognition accuracy.

## Main Research Question
How much labeled data can be reduced when using self-supervised pretraining for Arabic OCR?

## Planned Experiments
1. Train a supervised OCR baseline model.
2. Train a self-supervised/pretrained OCR model.
3. Compare both models using 100%, 50%, and 25% labeled data.
4. Evaluate results using Character Error Rate (CER) and Word Error Rate (WER).
